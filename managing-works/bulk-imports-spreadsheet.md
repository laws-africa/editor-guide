| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | - |

| country                           | <p>2-letter country code: <br><strong>ZA</strong> for South Africa, <strong>KE</strong> for Kenya</p>                                                                                                                                                                                                   | <p>Case insensitive<br>Must be the same code as the country you're in on the platform</p>                                                                                                                                                                                                                                                                                                       |   |
|                                   |                                                                                                                                                                                                                                                                                                         |                                                                                                                                                                                                                                                                                                                                                                                                 |   |
| locality                          | <p>locality code: <br><strong>cpt</strong> for Cape Town in South Africa</p>                                                                                                                                                                                                                            | <p>Leave blank for national legislation<br>Case insensitive<br>Must be the same code as the locality you're in on the platform<a href="https://en.wikipedia.org/wiki/List_of_municipalities_in_South_Africa"><br></a><a href="https://en.wikipedia.org/wiki/List_of_municipalities_in_South_Africa">ZA municipalities on Wikipedia</a></p>                                                      |   |
| title                             | <p>The short title of the work: <br><strong>Criminal Procedure Act</strong></p>                                                                                                                                                                                                                         | May or may not include the year at the end â€“ house style decision                                                                                                                                                                                                                                                                                                                               |   |

| doctype                           | <p>The code for the work's type: <br><strong>act</strong>, <strong>statement</strong></p>                                                                                                                                                                                                               | In most places the default is 'act', and this column can be ignored                                                                                                                                                                                                                                                                                                                             |   |
| subtype                           | <p>The code for the work's subtype: <br><strong>ln</strong> for a Legal Notice, <strong>by-law</strong> for a By-law, <strong>p</strong> for a Proclamation, <strong>si</strong> for a Statutory Instrument</p>                                                                                         | <p>Case insensitive<br>Leave blank for Acts</p>                                                                                                                                                                                                                                                                                                                                                 |   |
| actor                             | <p>A code decided on internally for the author of a work: <br><strong>mepc</strong></p>                                                                                                                                                                                                                 | <p>No spaces allowed</p><p>Can mostly be ignored</p>                                                                                                                                                                                                                                                                                                                                            |   |
| number                            | <p>The number of the work: <br><strong>9</strong> for Act 9 of 2012, <strong>34</strong> for SI 34 of 2012</p>                                                                                                                                                                                          | Use words separated by hyphens for works without a number and by-laws                                                                                                                                                                                                                                                                                                                           |   |
| year                              | <p>The year of the work: <br><strong>2012</strong> for Act 9 of 2012</p>                                                                                                                                                                                                                                |                                                                                                                                                                                                                                                                                                                                                                                                 |   |
| **Publication details**           |                                                                                                                                                                                                                                                                                                         | Used to try and automatically link the relevant Gazette when creating the work on the platform                                                                                                                                                                                                                                                                                                  |   |
| publication\_name                 | <p>Usually the name of the Gazette: <br><strong>Eastern Cape Provincial Gazette</strong></p>                                                                                                                                                                                                            |                                                                                                                                                                                                                                                                                                                                                                                                 |   |
| publication\_number               | <p>The Gazette number: <br><strong>12345</strong> for Gazette 12345 of 1 March 2020</p>                                                                                                                                                                                                                 | Found on front cover of Gazette                                                                                                                                                                                                                                                                                                                                                                 |   |
| publication\_date                 | <p>The Gazette date: <br><strong>2020-03-01</strong> for Gazette 12345 of 1 March 2020</p>                                                                                                                                                                                                              | <p>Found on front cover of Gazette<br>Date format is yyyy-mm-dd<br>Typing in e.g. '1 Mar 20' should be automatically formatted by the spreadsheet</p>                                                                                                                                                                                                                                           |   |
| **Other relevant dates**          |                                                                                                                                                                                                                                                                                                         | <p>Date format is yyyy-mm-dd<br>Typing in e.g. '1 Mar 20' should be automatically formatted by the spreadsheet</p>                                                                                                                                                                                                                                                                              |   |
| assent\_date                      | The date on which the work was signed by the relevant authority                                                                                                                                                                                                                                         |                                                                                                                                                                                                                                                                                                                                                                                                 |   |
| commencement\_date                | The date on which the work came / will come into force                                                                                                                                                                                                                                                  | <p>If the work has more than one commencement date, enter the main one here<br></p>                                                                                                                                                                                                                                                                                                             |   |
| **Other info**                    |                                                                                                                                                                                                                                                                                                         |                                                                                                                                                                                                                                                                                                                                                                                                 |   |
| stub                              | <p>Mark this work as a stub, i.e. a work for which the content will not be pulled in: <br>Commencement notices, amending works, repeal notices (works without substantive provisions)</p>                                                                                                               | Leave empty for principal works                                                                                                                                                                                                                                                                                                                                                                 |   |
| taxonomy                          | <p>Optional taxonomy tag/s to be applied to the work, separated by ';'s: <br><strong>lawsafrica-special:COVID-19</strong> for the Laws.Africa Special Collection titled 'COVID-19'</p>                                                                                                                  | <p>Choose an item from the dropdown<br>The taxonomy must exist on the platform first</p>                                                                                                                                                                                                                                                                                                        |   |
| **Passive relationships**         |                                                                                                                                                                                                                                                                                                         | <p>To select a work's FRBR URI from the dropdown, start typing its title in the cell and the list of options will shrink<br>There's no need to record both sides of a relationship in the spreadsheet; use the one that works best for your workflow</p>                                                                                                                                        |   |
| primary\_work                     | The FRBR URI of this work's parent / primary work                                                                                                                                                                                                                                                       | Only use for subsidiary legislation (e.g. Regulations / SIs)                                                                                                                                                                                                                                                                                                                                    |   |
| commenced\_by                     | The FRBR URI of a work that commences this work                                                                                                                                                                                                                                                         | <p>The commencing work will get its own row in the spreadsheet<br>Leave blank if the work commences itself (either on publication or at another date)</p>                                                                                                                                                                                                                                       |   |
| commenced\_on\_date               | The date on which this work is commenced by the commencing work                                                                                                                                                                                                                                         | The commencing work will most often commence on publication, but may give a different commencement date for the work being commenced                                                                                                                                                                                                                                                            |   |
| amended\_by                       | The FRBR URI of a work that amends this work                                                                                                                                                                                                                                                            | The amending work will get its own row in the spreadsheet                                                                                                                                                                                                                                                                                                                                       |   |
| amended\_on\_date                 | The date on which this work is amended by the amending work                                                                                                                                                                                                                                             | If nothing is given here, the commencement date of the amending work will be used (if it has one)                                                                                                                                                                                                                                                                                               |   |
| repealed\_by                      | The FRBR URI of a work that repeals this work                                                                                                                                                                                                                                                           | The repealing work will get its own row in the spreadsheet                                                                                                                                                                                                                                                                                                                                      |   |
| repealed\_on\_date                | The date on which this work is repealed by the repealing work                                                                                                                                                                                                                                           | If nothing is given here, the commencement date of the repealing work will be used (if it has one)                                                                                                                                                                                                                                                                                              |   |
| **Active relationships**          |                                                                                                                                                                                                                                                                                                         | <p>To select a work's FRBR URI from the dropdown, start typing its title in the cell and the list of options will shrink<br>There's no need to record both sides of a relationship in the spreadsheet; use the one that works best for your workflow</p>                                                                                                                                        |   |
| subleg                            | The FRBR URI of this work's children / subsidiary works, separated by ';'s                                                                                                                                                                                                                              |                                                                                                                                                                                                                                                                                                                                                                                                 |   |
| commences                         | The FRBR URI of a work commenced by this work                                                                                                                                                                                                                                                           | The work being commenced will get its own row in the spreadsheet                                                                                                                                                                                                                                                                                                                                |   |
| commences\_on\_date               | The date on which that work is commenced by this work                                                                                                                                                                                                                                                   | <p>The commencing work (this work) will most often commence on publication, but may give a different commencement date for the work being commenced </p><p>If nothing is given here, the commencement date of this work will be used (if it has one)</p>                                                                                                                                        |   |
| amends                            | The FRBR URI of a work amended by this work                                                                                                                                                                                                                                                             | <p>The amended work will get its own row in the spreadsheet<br>If this work amends more than one work, duplicate this work's row as many times as needed, only changing this column (and the next one if relevant)</p>                                                                                                                                                                          |   |
| amends\_on\_date                  | The date on which that work is amended by this work                                                                                                                                                                                                                                                     | If nothing is given here, the commencement date of this work will be used (if it has one)                                                                                                                                                                                                                                                                                                       |   |
| repeals                           | The FRBR URI of a work repealed by this work                                                                                                                                                                                                                                                            | <p>The repealed work will get its own row in the spreadsheet<br>If this work repeals more than one work, duplicate this work's row as many times as needed, only changing this column (and the next one if relevant)</p>                                                                                                                                                                        |   |
| repeals\_on\_date                 | The date on which that work is repealed by this work                                                                                                                                                                                                                                                    | If nothing is given here, the commencement date of this work will be used (if it has one)                                                                                                                                                                                                                                                                                                       |   |
