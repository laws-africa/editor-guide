# Bulk imports (spreadsheet)

The spreadsheet has the following columns:

| Column                            | Description                                                                                                                                                                                                                                                                                             | Examples                                                                                        | Notes                                                                                                                                                                                                                                                                                                                                                                                           |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Core details**                  |                                                                                                                                                                                                                                                                                                         |                                                                                                 | <p>Used for working out the FRBR URI (unique identifier)<br>Most fields are mandatory</p>                                                                                                                                                                                                                                                                                                       |
| country                           | 2-letter country code                                                                                                                                                                                                                                                                                   | ZA for South Africa, KE for Kenya                                                               | <p>Case insensitive<br>Must be the same code as the country you're in on the platform</p>                                                                                                                                                                                                                                                                                                       |
| locality                          | locality code                                                                                                                                                                                                                                                                                           | cpt for Cape Town in South Africa                                                               | <p>Leave blank for national legislation<br>Case insensitive<br>Must be the same code as the locality you're in on the platform<a href="https://en.wikipedia.org/wiki/List_of_municipalities_in_South_Africa"><br></a><a href="https://en.wikipedia.org/wiki/List_of_municipalities_in_South_Africa">ZA municipalities on Wikipedia</a></p>                                                      |
| title                             | The short title of the work                                                                                                                                                                                                                                                                             | Criminal Procedure Act                                                                          | May or may not include the year at the end – house style decision                                                                                                                                                                                                                                                                                                                               |
| cap                               | Chapter (Cap.) number                                                                                                                                                                                                                                                                                   | 12 for Chapter 12                                                                               | Not all countries use Chapter numbers                                                                                                                                                                                                                                                                                                                                                           |
| doctype                           | The code for the work's type                                                                                                                                                                                                                                                                            | act, statement                                                                                  | In most places the default is 'act', and this column can be ignored                                                                                                                                                                                                                                                                                                                             |
| subtype                           | The code for the work's subtype                                                                                                                                                                                                                                                                         | ln for a Legal Notice, by-law for a By-law, p for a Proclamation, si for a Statutory Instrument | <p>Case insensitive<br>Leave blank for Acts</p>                                                                                                                                                                                                                                                                                                                                                 |
| actor                             | A code decided on internally for the author of a work                                                                                                                                                                                                                                                   | mepc                                                                                            | <p>No spaces allowed</p><p>Can mostly be ignored</p>                                                                                                                                                                                                                                                                                                                                            |
| number                            | The number of the work                                                                                                                                                                                                                                                                                  | 9 for Act 9 of 2012, 34 for SI 34 of 2012                                                       | Use words separated by hyphens for works without a number and by-laws                                                                                                                                                                                                                                                                                                                           |
| year                              | The year of the work                                                                                                                                                                                                                                                                                    | 2012 for Act 9 of 2012                                                                          |                                                                                                                                                                                                                                                                                                                                                                                                 |
| **Publication details**           |                                                                                                                                                                                                                                                                                                         |                                                                                                 | Used to try and automatically link the relevant Gazette when creating the work on the platform                                                                                                                                                                                                                                                                                                  |
| publication\_name                 | Usually the name of the Gazette                                                                                                                                                                                                                                                                         | Eastern Cape Provincial Gazette                                                                 |                                                                                                                                                                                                                                                                                                                                                                                                 |
| publication\_number               | The Gazette number                                                                                                                                                                                                                                                                                      | 12345 for Gazette 12345 of 1 March 2020                                                         | Found on front cover of Gazette                                                                                                                                                                                                                                                                                                                                                                 |
| publication\_date                 | The Gazette date                                                                                                                                                                                                                                                                                        | 2020-03-01 for Gazette 12345 of 1 March 2020                                                    | <p>Found on front cover of Gazette<br>Date format is yyyy-mm-dd<br>Typing in e.g. '1 Mar 20' should be automatically formatted by the spreadsheet</p>                                                                                                                                                                                                                                           |
| **Other relevant dates**          |                                                                                                                                                                                                                                                                                                         |                                                                                                 | <p>Date format is yyyy-mm-dd<br>Typing in e.g. '1 Mar 20' should be automatically formatted by the spreadsheet</p>                                                                                                                                                                                                                                                                              |
| assent\_date                      | The date on which the work was signed by the relevant authority                                                                                                                                                                                                                                         |                                                                                                 |                                                                                                                                                                                                                                                                                                                                                                                                 |
| commencement\_date                | The date on which the work came / will come into force                                                                                                                                                                                                                                                  |                                                                                                 | <p>If the work has more than one commencement date, enter the main one here<br></p>                                                                                                                                                                                                                                                                                                             |
| **Other info**                    |                                                                                                                                                                                                                                                                                                         |                                                                                                 |                                                                                                                                                                                                                                                                                                                                                                                                 |
| stub                              | Mark this work as a stub, i.e. a work for which the content will not be pulled in                                                                                                                                                                                                                       | Commencement notices, amending works, repeal notices (works without substantive provisions)     | Leave empty for principal works                                                                                                                                                                                                                                                                                                                                                                 |
| taxonomy                          | Optional taxonomy tag/s to be applied to the work, separated by ';'s                                                                                                                                                                                                                                    | lawsafrica-special:COVID-19 for the Laws.Africa Special Collection titled 'COVID-19'            | <p>Choose an item from the dropdown<br>The taxonomy must exist on the platform first</p>                                                                                                                                                                                                                                                                                                        |
| **Passive relationships**         |                                                                                                                                                                                                                                                                                                         |                                                                                                 | <p>To select a work's FRBR URI from the dropdown, start typing its title in the cell and the list of options will shrink<br>There's no need to record both sides of a relationship in the spreadsheet; use the one that works best for your workflow</p>                                                                                                                                        |
| primary\_work                     | The FRBR URI of this work's parent / primary work                                                                                                                                                                                                                                                       |                                                                                                 | Only use for subsidiary legislation (e.g. Regulations / SIs)                                                                                                                                                                                                                                                                                                                                    |
| commenced\_by                     | The FRBR URI of a work that commences this work                                                                                                                                                                                                                                                         |                                                                                                 | <p>The commencing work will get its own row in the spreadsheet<br>Leave blank if the work commences itself (either on publication or at another date)</p>                                                                                                                                                                                                                                       |
| commenced\_on\_date               | The date on which this work is commenced by the commencing work                                                                                                                                                                                                                                         |                                                                                                 | The commencing work will most often commence on publication, but may give a different commencement date for the work being commenced                                                                                                                                                                                                                                                            |
| amended\_by                       | The FRBR URI of a work that amends this work                                                                                                                                                                                                                                                            |                                                                                                 | The amending work will get its own row in the spreadsheet                                                                                                                                                                                                                                                                                                                                       |
| amended\_on\_date                 | The date on which this work is amended by the amending work                                                                                                                                                                                                                                             |                                                                                                 | If nothing is given here, the commencement date of the amending work will be used (if it has one)                                                                                                                                                                                                                                                                                               |
| repealed\_by                      | The FRBR URI of a work that repeals this work                                                                                                                                                                                                                                                           |                                                                                                 | The repealing work will get its own row in the spreadsheet                                                                                                                                                                                                                                                                                                                                      |
| repealed\_on\_date                | The date on which this work is repealed by the repealing work                                                                                                                                                                                                                                           |                                                                                                 | If nothing is given here, the commencement date of the repealing work will be used (if it has one)                                                                                                                                                                                                                                                                                              |
| **Active relationships**          |                                                                                                                                                                                                                                                                                                         |                                                                                                 | <p>To select a work's FRBR URI from the dropdown, start typing its title in the cell and the list of options will shrink<br>There's no need to record both sides of a relationship in the spreadsheet; use the one that works best for your workflow</p>                                                                                                                                        |
| subleg                            | The FRBR URI of this work's children / subsidiary works, separated by ';'s                                                                                                                                                                                                                              |                                                                                                 |                                                                                                                                                                                                                                                                                                                                                                                                 |
| commences                         | The FRBR URI of a work commenced by this work                                                                                                                                                                                                                                                           |                                                                                                 | The work being commenced will get its own row in the spreadsheet                                                                                                                                                                                                                                                                                                                                |
| commences\_on\_date               | The date on which that work is commenced by this work                                                                                                                                                                                                                                                   |                                                                                                 | <p>The commencing work (this work) will most often commence on publication, but may give a different commencement date for the work being commenced </p><p>If nothing is given here, the commencement date of this work will be used (if it has one)</p>                                                                                                                                        |
| amends                            | The FRBR URI of a work amended by this work                                                                                                                                                                                                                                                             |                                                                                                 | <p>The amended work will get its own row in the spreadsheet<br>If this work amends more than one work, duplicate this work's row as many times as needed, only changing this column (and the next one if relevant)</p>                                                                                                                                                                          |
| amends\_on\_date                  | The date on which that work is amended by this work                                                                                                                                                                                                                                                     |                                                                                                 | If nothing is given here, the commencement date of this work will be used (if it has one)                                                                                                                                                                                                                                                                                                       |
| repeals                           | The FRBR URI of a work repealed by this work                                                                                                                                                                                                                                                            |                                                                                                 | <p>The repealed work will get its own row in the spreadsheet<br>If this work repeals more than one work, duplicate this work's row as many times as needed, only changing this column (and the next one if relevant)</p>                                                                                                                                                                        |
| repeals\_on\_date                 | The date on which that work is repealed by this work                                                                                                                                                                                                                                                    |                                                                                                 | If nothing is given here, the commencement date of this work will be used (if it has one)                                                                                                                                                                                                                                                                                                       |
| **Housekeeping**                  | ****                                                                                                                                                                                                                                                                                                    |                                                                                                 |                                                                                                                                                                                                                                                                                                                                                                                                 |
| Ignore                            | When doing a bulk import, put anything in this column and the importer will skip it. Putting a ✔ will make the cell green, to show it's been imported. Putting a "?" will make it yellow, to show research must still be done. It will be red as long as it's empty, to show it must still be imported. |                                                                                                 | <p>If a work already exists and is reimported ('Ignore' is left blank), it won't be overwritten on import and will be shown as a 'Duplicate', but the taxonomy and any relationships will be linked if they weren't linked previously </p><p>If any other non-core details were corrected on the platform in the meantime, they should be updated on the spreadsheet before it's reimported</p> |
| frbr\_uri                         | A formula in this cell will generate the FRBR URI                                                                                                                                                                                                                                                       | /akn/za/act/2020/1 for Act 1 of 2020 in South Africa                                            | <p>Do not edit this cell<br>If it looks wrong, check the core details</p>                                                                                                                                                                                                                                                                                                                       |
| frbr\_uri\_title                  | A formula in this cell will generate the FRBR URI followed by the title                                                                                                                                                                                                                                 | /akn/za/act/2020/1 - Banking Act, 2020 for Act 1 of 2020 in South Africa                        | This populates the dropdown lists for all relationships                                                                                                                                                                                                                                                                                                                                         |
| **Comments etc**                  |                                                                                                                                                                                                                                                                                                         |                                                                                                 |                                                                                                                                                                                                                                                                                                                                                                                                 |
| comments                          | Text from this column and any others you add won't be imported; use them as a record for yourself                                                                                                                                                                                                       |                                                                                                 |                                                                                                                                                                                                                                                                                                                                                                                                 |
| LINKS ETC (add columns as needed) |                                                                                                                                                                                                                                                                                                         |                                                                                                 |                                                                                                                                                                                                                                                                                                                                                                                                 |
